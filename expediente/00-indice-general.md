## Guía de Lectura Recomendada

### Para Nuevos Usuarios / Capacitación
1. Leer **01 - Documentación de Inicio** → Entender qué es el sistema
2. Leer **02 - Flujo de la Aplicación** → Conocer su rol y flujo de trabajo
3. Leer **06 - Glosario** → Familiarizarse con términos

### Para Desarrolladores
1. Leer **01 - Documentación de Inicio** → Contexto general
2. Leer **04 - Documentación Técnica** → Arquitectura y detalles técnicos
3. Leer **05 - Documentación de Seguridad** → Buenas prácticas
4. Leer **06 - Glosario** → Referencia rápida

### Para Administradores del Sistema
1. Leer **01 - Documentación de Inicio** → Setup y configuración
2. Leer **03 - Procesos por Rol** → Endpoints y capacidades
3. Leer **05 - Documentación de Seguridad** → Hardening y recomendaciones
4. Leer **06 - Glosario** → Checklist de producción

### Para Auditores
1. Leer **02 - Flujo de la Aplicación** → Quién hace qué
2. Leer **03 - Procesos por Rol** → Matriz de permisos
3. Leer **05 - Documentación de Seguridad** → Controles y vulnerabilidades
